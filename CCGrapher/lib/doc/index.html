derive (f, delta)
Create a derivative for a function This function takes two arguments, a function and a delta. The delta can be left unspecified. The function returned is an approximation for the derivative of the given function, and delta specifies the accuracy, where a smaller delta results in a higher accuracy.

Parameters:

  • f func The function to derive from
  • delta number The accuracy of the derivation, defaults to 1e-4

Returns:

    func The derivative of f
